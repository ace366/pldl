- 2026-03-20

## checkpoint

- Git tag: `ai-checkpoint-20260320-admin-mobile-nav`
- Base commit: `976c1b7`

## 依頼概要

- 管理者ログイン時だけ、スマホ上部ヘッダーを管理者向けに最適化する
- ロゴを中央に寄せ、3本線メニューを左端に移動する
- ChatGPT 風の左スライドサイドバーで、PC表示中の管理者メニューをスマホから使いやすく再配置する
- 既存のスマホ下メニューは残し、権限制御はそのまま維持する

## 実装方針

- 共通ナビ `resources/views/layouts/navigation.blade.php` のみで吸収し、他ロールのヘッダーは原則維持
- 管理者スマホ時のみ左ハンバーガー + 中央ロゴの専用ヘッダーを表示
- 既存の `permCan(...)` と `Route::has(...)` を再利用し、権限のある項目だけをサイドバーへ表示
- サイドバー内は `おしらせ / 基本導線`、`勤怠管理`、`運営管理`、`アカウント` に整理し、折りたたみ式にする

## 変更ファイル

- `resources/views/layouts/navigation.blade.php`

## 確認コマンド

- `php artisan view:cache`

## 確認メモ

- 管理者スマホ時だけ、上部ヘッダーが左ハンバーガー・中央ロゴ構成になる
- サイドバーは左スライドのオーバーレイ形式で開閉する
- スマホ下固定メニューは従来どおり残る
- family および非管理者のスマホ上部ヘッダーは既存動作を維持する
