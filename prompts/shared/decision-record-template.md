# Decision Record Template

以下を `docs/decisions/` に保存して使う。

```md
# Date

YYYY-MM-DD

# Title

判断タイトルを記載

# 背景

- どの機能・運用・制約に対する判断か
- 何が問題だったか

# 決定内容

- 何を採用したか
- 何を採用しなかったか

# 理由

- なぜその判断にしたか
- 既存構造や運用上の理由

# 影響範囲

- routes
- controllers
- blade
- middleware
- DB
- 運用

# 今後の対応

- 将来見直す条件
- 後続タスク

# 備考

- 関連 Issue
- 関連 AI log
- 参考 commit
```
